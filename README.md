# DADL Registry

Community registry for [DADL](https://dadl.ai/spec) files — declarative API descriptions for [ToolMesh](https://github.com/DunkelCloud/ToolMesh).

## Links

- **[Browse DADLs](https://dadl.ai/browse)** — interactive browser with full tool details
- **[DADL Spec](https://dadl.ai/spec)** — the format specification
- **[How to create a DADL](https://dadl.ai/guides/creating-a-dadl)** — step-by-step guide
- **[ToolMesh](https://github.com/DunkelCloud/ToolMesh)** — the secure MCP gateway that runs these DADLs

## Contributing

Want to add a DADL for an API? Fork this repo, add your `.dadl` file, and open a PR. See the [creation guide](https://dadl.ai/guides/creating-a-dadl) for the format and validation rules.
